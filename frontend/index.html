About This Tool

This is an AI-powered research prototype for predicting hearing deficiency risk in newborns. It combines genomic and clinical data using multiple machine learning models including:

  • Random Forest
  • Support Vector Machine (SVM)
  • XGBoost/Gradient Boosting
  • Artificial Neural Network (ANN)
  • Transformer-based classifier

The models are explained using SHAP (SHapley Additive exPlanations) to provide interpretable insights into risk factors. This tool uses synthetic data and is intended for research and educational purposes only.

Key Features:

  • Multi-model ensemble prediction
  • SHAP-based explainability
  • Clinical and genomic feature integration
  • Batch processing support
  • Risk stratification (Low/Medium/High)
